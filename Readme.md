# PASSIFY - Visitor Pass Management System

**PASSIFY** 
A modern Pass Management System for security managment
---

## Key Features

###  For Visitors
*   **Self-Registration**: Visitors can easily Register themselves from the visitor dashboard.
*   **Appointment Scheduling**:Visitors can easily Schedule the Appointments wheen they want to.
*   **Digital Entry Pass**: Once their appointment is approved they with recieve a digital entry pass for their entry with QrCode.
*   **PDF Download**: Visitors can Easily dowload the pdf containing the qr code and pdf will be mailed to the visitors.

### For Employees
*   **Actionable Dashboard**: employees can approve and reject the appointments.
*   **Historical Log**: They Can also track Past appointments.

### For Security
*   **QR Scanner Terminal**: Security Personnel can scan the qr code at the time of the entry.

### For Administrators
*   **User Onboarding**: Admin can manage the staff accounts and if some violation is occurred admin can disable the staff account.
*   **Visitor Directory**: Admin can also see the visitors who regisitered.
